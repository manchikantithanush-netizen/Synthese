
# Synthese

Synthese
An all-in-one sports application designed for young athletes. Synthese provides a comprehensive platform to track performance, manage training, and connect with the athletic community, utilizing a freemium financial model.





## Tech Stack

**Framework:** Flutter

**Language:** Dart

**Backend:** Firebase & Firestore

**AI Integration:** Llama (Via GitHub Models)



## Features

- Performance Tracking: Log and analyze athletic stats in real-time.

- Various Trackers: Finance, Cycles, Mindfulness, Diet etc.

- Freemium Access: Core features available for all users, with premium enhancements for advanced athletes.

- Seamless Sync: Cloud-powered data management via Firestore


## Run Locally

Clone the project

```bash
  git clone https://github.com/manchikantithanush-netizen/synthese.git
```

Go to the project directory

```bash
  cd synthese
```

Install dependencies

```bash
  flutter pub get
```

Start the server

```bash
  flutter run
```
